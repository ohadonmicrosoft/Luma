# Luma Design System

The Luma design system provides a comprehensive set of design tokens, components, and patterns for building consistent user interfaces across the Luma e-commerce platform.

## Design Tokens

Design tokens are the visual design atoms of the design system — specifically, they are named entities that store visual design attributes. We use them in place of hard-coded values to ensure a scalable and consistent visual system.

## Implementation

Our design tokens are implemented as TypeScript objects in the `frontend/src/styles/tokens` directory:

```typescript
// Import the entire token system
import tokens from '@/styles/tokens';

// Import specific token categories
import { colors, typography, spacingTokens, elevationTokens } from '@/styles/tokens';

// Import specific tokens from a category
import { primary, secondary } from '@/styles/tokens/colors';
```

These tokens are then integrated with Tailwind CSS through the `tailwind.config.js` file, making them available as utility classes:

```jsx
// Using color tokens
<div className="bg-primary-600 text-white">Primary Button</div>

// Using typography tokens
<h1 className="font-sans text-3xl font-bold leading-tight">Heading</h1>

// Using spacing tokens
<div className="p-4 m-6">Spaced Content</div>

// Using elevation tokens
<div className="shadow-md rounded-lg">Elevated Card</div>
```
